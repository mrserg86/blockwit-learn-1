1. Дерево - структура данных, в которой каждый узел имеет не более двух потомков. Данные - узлы(вершины) дерева.
Пример - используется для поиска в упорядоченном списке - поиск через деление пополам.

2. Список - это тип данных, в котором каждый элемент содержит помимо данных также ссылку на следующий элемент списка.
Пример - негры, висящие над обрывом...

3. Ассоциативный массив - это набор уникальных ключей и значений, где каждый ключ связан с одним значением.
Пример - телефонный справочник.

4. Очередь - структура данных, отражение реальной очереди людей в магаз. Главный принцип: первый зашёл - первый вышел;
последний зашёл - последний вышел.
Пример - очередь в магаз.

5. Стэк - очередь, работающая по принципу: первый зашёл - последний вышел и последний защёл - первый вышел.
Пример - куча мешков с картошкой. Сдача карт - снизу первые, сверху последние.


  FIFO - очередь короче.
  LIFO - элементы могут добавляться и выбираться только с одного конца, называемого «вершиной списка».
